# C++ Console Pokemon game

<br />

## Requirements

- C++ installed
- C++ compiler, `gcc` or `g++`

<br />

## Install and run

Download the repo locally.

Open terminal in the folder you downloaded the repo to.

```zsh
cd cpp-console-pokemon
```

```zsh
# First way
g++ main.cpp
./a.out

# Second way
./a.out

# Third way
./run.sh
```
